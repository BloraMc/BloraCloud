# BloraCloud

BloraCloud is a community-driven project hosting **editable cloud JSON/TXT indexes** and **Cloud Data**. It’s designed so anyone can contribute data, such as Minecraft servers, game stats, or other cloud resources.

Now, it has multiple JSON/TXT files like `mcjavasrv.json`, which stores **Minecraft Java server info**. Users can create **pull requests** to add more data to expand cloud info.
